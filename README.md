# Gnome Keymap 
A macOS + Hyprland alike keymap on Gnome/Nobara that should work on most Gnome-based DE.

### How to use
copy `user_gnome.backup` to your conf by : 

    cp user_gnome.backup ~/.config/dconf/user

backup the same way but so, the configured keymap become :
